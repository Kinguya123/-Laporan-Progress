# -Laporan-Progress
Aplikasi Monitoring dan Management Siswa merupakan aplikasi yang dapat memantau dan memonitoring siswa selama dalam masa Prakerin untuk mempermudah guru pembimbing, pembimbing perusahaan, dan Hubin dalam memantau perkembangan siswa ditempat Prakerin dan untuk mempermudah laporan kepada guru pembimbing, pembimbing perusahaan, dan Hubin.
